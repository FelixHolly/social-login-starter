Understanding the Data

Username: The principal name from the authentication object.

Authorities: Granted authorities/roles for access control. Spring Security automatically grants ROLE_USER to OAuth2 authenticated users.

OAuth2 Attributes: Raw data from the provider's userinfo endpoint. This includes user ID, email, name, profile picture, and other provider-specific fields.
